# CLI To-Do App

A simple command-line to-do list application for managing your tasks efficiently.

## Features

- Add, view, and delete tasks
- Mark tasks as complete
- Simple and intuitive CLI interface

## Installation

```bash
git clone <repository-url>

npm install
```

## Scripts

```bash
# build ts code into js
npm run build

# start app from
npm run start

# build ts code into js and start app
npm run build:start
```

## Usage

```bash
# when app is started type help to see a list of commands for navigating app
help
```
